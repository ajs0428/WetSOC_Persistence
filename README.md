## Abstract 

This repository contains analysis and data for a study that addresses a gap in soil science by examining mineral-associated organic carbon (MAOM-C) in forested wetlands. 



## Directory for Wetland SOC Persistence 
`/data`: processed data, mostly dataframes

- Raw data is stored on private local drives for now but processed data are available

`/analysis`: R scripts and Quarto notebooks for data analysis and processing 

`/writing`: Markdown/Quarto files for manuscript drafting
